🚀 Bienvenido a DocuWrite Pro

Este es un editor de documentos profesional y moderno con tema azul. Comienza a escribir tu contenido aquí.

✨ Nuevas características

  • 🎨 Tema azul profesional - Diseño moderno y elegante
  • 📄 Configuración de página - Orientación vertical/horizontal y tamaños (Carta, A4, Legal)
  • 🖼️ Multimedia - Inserta imágenes, videos y figuras con pie de foto
  • 👁️ Vistas múltiples - Modo edición, lectura y vista previa
  • 💬 Comentarios - Panel lateral para notas y revisiones
  • 🎯 Más formatos - Resaltados, sombras de texto y estilos avanzados
  • ⚡ Preparado para React - Estructura modular para migración futura
💡 Tip: Usa Ctrl+B para negrita, Ctrl+I para cursiva, Ctrl+K para enlaces y Ctrl+F para buscar.
